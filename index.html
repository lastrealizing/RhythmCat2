RhythmCat2

The next generation of RhythmCat Music Player.

Download this project as a .zip file Download this project as a tar.gz file

Introduction:

RhythmCat2 the next gernation of RhythmCat Music Player. It is a music player which can be used under Linux. It is not only a normal music player, it has both command-line interface (to be implemented) and graphic user interface, and it can extend its features by plug-ins, like lyric show in window, desktop lyric show, etc... The GTK+ UI can use custom GTK+ 3 themes, and the UI layout can be changed partially.

Since this project in just in experimental stage, only GTK+ UI frontend has been implemented.

How to install this program:

Get the latest version from Github:

git clone git://github.com/supercatexpert/RhythmCat2.git

Now only source code is available, you should compile it, then you can use this program. Before you compile it, you should check the developing environment, and the libraries which this program depends. On Debian/Ubuntu, you can run the commands below to help you install the packages that the compilation needed:

sudo apt-get install build-essential libgtk-3-dev libgstreamer0.10-dev \ libgstreamer-plugins-base0.10-dev libjson0-dev libcurl4-openssl-dev

If you want to compile Python3 loader and use Python3 plug-ins, install package python3-dev:

sudo apt-get install python3-dev

If you want to play MP3 or other Restricted Format, please install Gstreamer Bad & Ugly plugins. In Debian/Ubuntu, you can just run:

sudo apt-get install gstreamer0.10-plugins-bad gstreamer0.10-plugins-ugly

Then you can try to compile it:

./configure --with-native-plugins --with-python3-plugins

make

sudo make install

PS: You can remove --with-native-plugins or --with-python3-plugins arguments if you do not need native plug-ins or Python3 plug-ins.

then just run /usr/local/bin/RhythmCat2, feel happy with it!
